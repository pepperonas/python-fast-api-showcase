## Screenshot-Anleitung

### Mit Browser DevTools:
1. Öffne die App im Browser (http://localhost:3000)
2. Öffne DevTools (F12)
3. Aktiviere den Device Toolbar (Strg+Shift+M / Cmd+Shift+M)
4. Wähle eine Desktop-Auflösung (z.B. 1920x1080)
5. Navigiere zur gewünschten View
6. Verwende Browser-Erweiterungen wie "Full Page Screen Capture" oder ähnliche

### Mit Screenshot-Tools:
- **macOS**: Cmd+Shift+4 für Bereich, Cmd+Shift+3 für Vollbild
- **Windows**: Snipping Tool oder Win+Shift+S
- **Browser Extensions**: z.B. "Awesome Screenshot" oder "Nimbus Screenshot"

### Empfohlene Auflösung:
- **Desktop**: 1920x1080 oder 1440x900
- **Format**: PNG oder JPG
- **Qualität**: Hoch (für bessere Darstellung in der README)
